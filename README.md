# London 2026 Trip

Post-trip website with the itinerary, an interactive map, and photos from our UK trip (London, plus day trips to the Cotswolds and Cambridge).

## Structure (planned)
- `index.html` — itinerary + map (schedule entries link to photo locations)
- `gallery.html` (or a tab within `index.html`) — full photo gallery
- `photos/` — trip photos
- `data/photo_locations.csv` — filename, timestamp, GPS, and confirmed location for each photo

## Hosting
Served via GitHub Pages once content is in place.
